# Pinia

  1. Instalar Pinia

Lo instalo con yarn ya que con npm me dá problemas. Posiblemente tengas que instalar yarn búsca cómo instalarlo en tu sistema operatiro.

yarn install @pinia/nuxt

No me acuerdo si es necesario instalar también pinia

yarn install pinia
  1. Vete a nuxt.config.ts y añade:
modules: [
      '@pinia/nuxt',
    ],
