    # Supabase

    Supabase Nuxt 3 install (opens new window)

    # Instalación

    1. Instalar módulo
    yarn add --dev @nuxtjs/supabase
    1. Archivo nuxt.config.js añadir módulo
    '@nuxtjs/supabase'
    1. Crear archivo .env
    SUPABASE_URL="https://example.supabase.com"
SUPABASE_KEY="<supabase api-key>"
    1. Funciones CRUD, obtener, actualizar, eliminar
    <script setup>
  const supabase = useSupabaseClient()
</script>
    1. Funciones autentificación
    <script setup>
const user = useSupabaseUser();
</script>
